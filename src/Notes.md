*Сущности*
+ Анализатор игрового процесса - проверка состояния игры
+ Игра - игровой процесс
+ Игрок - хранение состояния выйгрыш\проигрыш + хранение введённого символа DONE
+ Ридер - читать словарь, картинки из файла DONE
+ Картинки - хранение картинок DONE
+ Словарь - хранение словаря DONE
+ Загаданное слово - хранение, состояние загаданного слова DONE
+ Генератор слова - выбор слова из словаря DONE
